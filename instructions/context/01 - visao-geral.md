# Visão Geral do Projeto

## Propósito

O projeto "Euaggelion" é uma aplicação web desenvolvida com Next.js, focada em fornecer conteúdo religioso e informativo. O nome "Euaggelion" é derivado do grego para "evangelho", refletindo o propósito do projeto de disseminar informações e ensinamentos religiosos.

## Objetivos

- Fornecer artigos e conteúdo religioso de forma acessível.
- Permitir a busca e navegação fácil por tópicos e categorias.
- Oferecer uma interface amigável e responsiva para os usuários.
- Integrar funcionalidades como webmentions e newsletters para engajamento.

## Público-Alvo

- Cristãos interessados em crescer espiritualmente.
- Usuários que buscam informações sobre fé, teologia e ensinamentos bíblicos.
- Comunidades que desejam compartilhar e discutir conteúdo religioso.

## Funcionalidades Principais

- Artigos e páginas informativas.
- Busca por conteúdo.
- Webmentions para interação social.
- Newsletter para atualizações.
- Navegação por categorias e tags.

## Tecnologias Base

- Next.js: Framework principal para desenvolvimento web.
- React: Biblioteca para construção de interfaces.
- TypeScript: Linguagem de programação para tipagem estática.
- MDX: Formato para conteúdo rico em markdown.
- Tailwind CSS: Framework para estilos e design.
